## How to use
first you have to install `pdm` `conda` or `pip` to manage this project

#### pdm
```shell
pip install --user pdm
pdm install
pdm run server
```

#### pip
```shell
pip install -r requirements.txt
```

#### conda
```shell
conda activate -your env-
pip install -r requirements.txt
```
